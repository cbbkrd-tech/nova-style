Lista serwisów
Struktura
Zasób Service posiada następujące atrybuty:

Atrybut

Typ

Opis

id

String

Unikalny identyfikator serwisu

name

String

Nazwa serwisu

description

String

Opis serwisu

additional_services

Array

Lista dostępnych usług dodatkowych. 




Pole additional_services posiada następujące atrybuty:

Atrybut

Typ

Opis

id

String

Unikalny identyfikator

name

String

Nazwa

description

String

Opis
